# 🚀 AI & Robotics Workshop Website

A full-stack web application built using the MERN stack that allows users to explore and enroll in an AI & Robotics workshop. The project features a modern UI, smooth user experience, and a functional backend with database integration.

---

## 📌 Features

* 🎯 Attractive Hero Section with modern UI
* 📜 Smooth scrolling to enquiry section
* 📝 Enquiry Form for user registration
* 🔗 Frontend to Backend API integration
* 💾 Data stored in MongoDB database
* ⚡ Responsive design for all devices

---

## 🛠️ Tech Stack

### Frontend:

* React.js
* Tailwind CSS
* React Router

### Backend:

* Node.js
* Express.js

### Database:

* MongoDB (MongoDB Atlas)

## ⚙️ Installation & Setup

### 1️⃣ Clone the Repository

```bash
git clone https://github.com/Pradip-ux/workshop-project.git
cd workshop-project
```

---

### 2️⃣ Setup Backend

```bash
cd backend
npm install
```

Create a `.env` file in backend folder:

```
MONGO_URI=your_mongodb_connection_string
PORT=5000
```

Run backend server:

```bash
npm start
```

---

### 3️⃣ Setup Frontend

```bash
cd frontend
npm install
npm start
```
